What is Linux ?

Like windows 7,8,10 or iOS (MacOS) its an operating system, BUT FOR FREE

“Free software” means software that respects users' freedom and community, meaning that the users have the freedom to run, copy, distribute, study, change and improve the software. Thus, “free software” is a matter of liberty, not price. To understand the concept, you should think of “free” as in “free speech,” not as in “free beer”.

Web Server/ HTTP server

A Web server is a program that uses HTTP (Hypertext Transfer Protocol) to serve the files that form Web pages to users, in response to their requests, which are forwarded by their computers' HTTP clients.

FTP server

An FTP server is a computer which has a file transfer protocol (FTP) address and is dedicated to receiving an FTP connection. As name suggest its used for sharing small files and folders.

Storage server

A storage server is a type of server that is used to store, access, secure and manage digital data, files and services. It is a purpose built server used for storing and accessing small to large amount of data over a shared network or through the Internet.A storage server may also be called a file server.

Directory server

A Directory Server can be used for storing identity profiles and access privileges to information about application and network resources.Makes use of authorization of users to enable secure access to enterprise and Internet services.

Database server

A database server is a server which houses a database application that provides database services to other computer programs which demands for the flow of data in either direction that be database or application.

Mail server

A mail server/e-mail server is a server that handles and delivers e-mail over a network. A mail server can receive e-mails from client computers and deliver them to other mail servers, but can also be centrally managed within an organization.

Proxy server

A Proxy server is a server that acts as an intermediary for requests from clients seeking resources from other servers. A client connects to the proxy server, requesting some service, such as a file, connection, web page, or other resource available from a different servers. Web proxies facilitates access to content on the World Wide Web, providing anonymity and may be used to bypass IP address blocking.

Monitoring Server

Server monitoring is the process of reviewing and analyzing a server for availability, operations, performance, security and other operations-related processes.It is performed by server administrators to ensure that the server is performing as expected and to mitigate problems as they become apparent.
